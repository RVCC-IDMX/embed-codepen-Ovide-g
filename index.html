Image Text Reveal Hover Effect

by Garry Ovide

The author uses an image and styles it in this pen. A transform scale is employed to resize the element and a transform origin is used to apply a transformation. Also, apply Transition to control the animation speed that allows property values to be changed smoothly. In order to achieve transparency on the image, she sets the opacity and reveals the text with the hoover effect.

See the Pen Image Text Reveal Hover Effect by Ritika Agrawal (@RitikaAgrawal08) on CodePen.
